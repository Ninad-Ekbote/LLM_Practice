# LLM_Practice
This repository contains my hands-on practice and experiments with various **Large Language Models (LLMs)** from scratch. I won't be using this repo to train models rather just a way to showcase my learning

## Purpose
The goal of this repository is to document and organize my learning while working with different LLM architectures, techniques, and implementations. It serves as a personal reference for concepts, experiments, and implementations related to modern language models.

## What This Repository Includes
- Implementations of different LLM components
- Experiments with model architectures
- Training and fine-tuning practices
- Notes and code related to LLM concepts
- Small projects and exploratory work

## Topics Covered/ I plan to cover
Some of the areas explored in this repository include:
- Transformer architecture
- Attention mechanisms
- KV Caching
- Tokenization
- Training pipelines
- Fine-tuning techniques
- Optimization and inference strategies

## Goal
To build a strong practical understanding of how modern LLM systems work by implementing and experimenting with their core components.
